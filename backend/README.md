# Meat Shop Backend API

Backend API для мясного магазина с централизованным хранением заказов и чатов.

## Установка

1. Перейдите в папку backend:
```bash
cd backend
```

2. Установите зависимости:
```bash
npm install
```

## Запуск

Запустите сервер:
```bash
npm start
```

Сервер будет доступен по адресу: `http://localhost:5000`

## API Endpoints

### Заказы

- `GET /api/orders` - Получить все заказы
- `GET /api/orders/:id` - Получить заказ по ID
- `POST /api/orders` - Создать новый заказ

### Сообщения

- `GET /api/orders/:orderId/messages` - Получить все сообщения по заказу
- `POST /api/orders/:orderId/messages` - Отправить сообщение (поддерживает файлы)

### Система

- `GET /api/health` - Проверка здоровья сервера

## Функциональность

- ✅ Создание заказов
- ✅ Получение всех заказов
- ✅ Получение одного заказа
- ✅ Отправка сообщений с текстом
- ✅ Отправка сообщений с медиа-файлами (фото/видео)
- ✅ Real-time обновления через Socket.io
- ✅ Загрузка файлов (макс. 20MB)
- ✅ CORS для фронтенда

## Хранение данных

В текущей версии данные хранятся в памяти (in-memory). При перезапуске сервера данные будут сброшены.

Для продакшена рекомендуется использовать:
- PostgreSQL
- MongoDB
- или файловое хранилище (db.json)

# Файлы чата в админ-панели

## Структура файлов чата

### Основные компоненты

#### 1. `meat-shop-react/src/components/AdminChatWidget.jsx`
**Новый компонент чата для админ-панели**

- **Функционал**: Полностью копирует функционал пользовательского чата-виджета
- **Особенности**:
  - Плавающая кнопка чата в правом нижнем углу экрана
  - Модальное окно с чатом
  - Отправка текстовых сообщений
  - Отправка фото и видео файлов (до 5MB для фото, 20MB для видео)
  - Автоматическая загрузка сообщений
  - Real-time polling каждые 3 секунды
  - Автоматическое открытие чата при получении сообщения от покупателя
  - Плавные анимации и современный дизайн
  - Предпросмотр загружаемых файлов
  - Защита от прокрутки страницы при открытом чате
  - Поддержка форматов: JPEG, PNG, GIF, WebP, MP4, WebM

- **API**: Использует те же функции из `../utils/api.js`:
  - `fetchMessages(orderId)` - получение сообщений
  - `sendMessage(orderId, senderId, senderType, text, mediaFile)` - отправка сообщения

- **CSS**: Использует стили из `./ChatWidget.css` (тот же файл, что и у пользователя)

- **Параметры компонента**:
  - `orderId` (number) - ID заказа
  - `currentUserId` (number) - ID текущего пользователя (администратора)
  - `isAdmin` (boolean) - флаг, что это админ-панель (по умолчанию true)

#### 2. `meat-shop-react/src/components/AdminPanel.jsx`
**Админ-панель с интеграцией чата**

- **Изменения**:
  - Импортирован новый компонент: `import AdminChatWidget from './AdminChatWidget';`
  - Заменен импорт старого компонента: `import OrderChat from './OrderChat';`
  - В модальном окне чата используется новый компонент:
    ```jsx
    <AdminChatWidget
      orderId={selectedOrder.id}
      currentUserId={1} // Admin user ID
      isAdmin={true}
    />
    ```

- **Функции для работы с чатом**:
  - `handleOpenChat(orderId)` - открытие модального окна с чатом
  - `handleCloseChat()` - закрытие модального окна

#### 3. `meat-shop-react/src/components/ChatWidget.css`
**CSS стили для чата**

- **Используется**: Общий файл стилей для обоих чатов (пользовательского и админского)
- **Стили**:
  - `.chat-widget` - контейнер кнопки чата
  - `.chat-widget-toggle` - кнопка открытия чата
  - `.chat-widget-content` - модальное окно чата
  - `.chat-widget-header` - шапка чата
  - `.chat-widget-messages` - область сообщений
  - `.message` - базовый блок сообщения
  - `.message.admin` - сообщение администратора (справа)
  - `.message.customer` - сообщение покупателя (слева)
  - `.chat-widget-input-form` - форма ввода
  - `.file-input-wrapper` - блок выбора файла
  - `.file-preview` - превью загруженного файла
  - `.chat-error` - блок ошибок
  - Анимации: `slideUpFade`, `messagePop`, `pulse`

#### 4. `meat-shop-react/src/components/ChatWidget.jsx`
**Пользовательский чат-виджет** (оригинал)

- **Используется**: Для чата на странице заказа у пользователей
- **Функционал**: Тот же, что и в AdminChatWidget, но для пользователей
- **Отличия**:
  - Определяет orderId из URL страницы `/order/:id`
  - Отправляет сообщения от типа 'customer'
  - Автоматически открывает чат при получении сообщения от продавца (admin)

#### 5. `meat-shop-react/src/utils/api.js`
**API утилиты для общения с бэкендом**

- **Функции**:
  - `fetchMessages(orderId)` - получение сообщений для заказа
  - `sendMessage(orderId, senderId, senderType, text, mediaFile)` - отправка сообщения
  - `createOrder(orderData)` - создание заказа
  - `fetchOrders()` - получение списка заказов
  - `fetchOrder(orderId)` - получение одного заказа
  - `checkHealth()` - проверка здоровья сервера

#### 6. `backend/server.js`
**Backend сервер**

- **API endpoints**:
  - `GET /api/orders` - получение всех заказов
  - `GET /api/orders/:id` - получение одного заказа
  - `POST /api/orders` - создание заказа
  - `GET /api/orders/:id/messages` - получение сообщений заказа
  - `POST /api/orders/:id/messages` - отправка сообщения
  - `GET /api/health` - проверка здоровья сервера

### Старый компонент (не используется)

#### 7. `meat-shop-react/src/components/OrderChat.jsx`
**Старый компонент чата для админ-панели**

- **Статус**: Не используется, заменен на AdminChatWidget
- **Оставлен для**: Возможности быстрого возврата к старой версии

## Как использовать

### В админ-панели:

1. Зайдите в админ-панель (пароль: admin123)
2. Перейдите на вкладку "Заказы"
3. Нажмите "👁 Подробнее" на любом заказе
4. Нажмите кнопку "💬 Открыть чат"
5. В правом нижнем углу появится плавающая кнопка чата
6. Нажмите на неё, чтобы открыть чат
7. Отправляйте сообщения, прикрепляйте фото/видео

### В пользовательской части:

1. Зайдите на страницу заказа `/order/:id`
2. В правом нижнем углу появится плавающая кнопка чата
3. Нажмите на неё, чтобы открыть чат
4. Отправляйте сообщения, прикрепляйте фото/видео

## Технические детали

### Определение orderId

Компонент AdminChatWidget определяет orderId в таком порядке приоритета:
1. Из пропсов (которые передаются из AdminPanel)
2. Из URL страницы `/admin/order/:id`
3. Из localStorage

### Real-time обновление

- Используется polling каждые 3 секунды
- При получении нового сообщения автоматически открывается чат (если он закрыт)
- Автоскролл к последнему сообщению

### Форматы файлов

**Изображения**:
- JPEG, PNG, GIF, WebP
- Максимум: 5MB

**Видео**:
- MP4, WebM
- Максимум: 20MB

### Отправка сообщений

**Текстовые сообщения**:
```javascript
await sendMessage(orderId, senderId, 'admin', 'Текст сообщения');
```

**С медиа**:
```javascript
await sendMessage(orderId, senderId, 'admin', 'Текст сообщения', file);
```
